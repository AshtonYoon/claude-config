# Claude 글로벌 설정 공유

## 설명
개인 PC 여러 대에 동일한 Claude 글로벌 설정을 공유하기 위한 레포지토리입니다. (`~/.claude` 설정용)

## 사전 준비
- **uv** 설치 필요
